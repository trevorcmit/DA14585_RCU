#### DA14585 RCU

Hello! This repository sets up an environment for you to build the "DA14585 Voice RCU" code outside of Keil uVision environment. The original example code is setup solely for usage with the proprietary Voice RCU remote, app, and flashable example software. Our goal here is to give you a means to edit and actively debug the Voice RCU demo on the dev kit itself or other implementations of the DA14585 such as within a custom PCB.
